📝 STATE MANAGEMENT

All form state is managed in App.jsx using React hooks (useState):
  • formData: Stores all form inputs
  • errors: Stores validation errors
  • currentStep: Tracks which form to display

Data flows:
  ↓ FROM App.jsx (Parent)
  → TO Components (Children) as Props
  ↑ FROM Components (Children)
  → TO App.jsx via Callback Functions

🔄 FORM WORKFLOW

STEP 1: APPLICATION FORM
├─ Fill personal information
├─ Add minimum 1 education entry
├─ Optionally add work experience
├─ Upload required resume file
├─ Click "PROCEED"
└─ Validation runs → Success? Go to Step 2 : Show errors

STEP 2: REVIEW FORM
├─ Display all entered information
├─ User review data
├─ Two options:
│  ├─ "BACK" → Return to Step 1 to edit (data preserved)
│  └─ "SUBMIT" → Proceed to Step 3
│
STEP 3: CONFIRMATION FORM
├─ Show success message
├─ Display next steps
└─ "RETURN HOME" → Reset form and go back to Step 1

→ Install Node.js from https://nodejs.org

Issue: Port 3000 already in use
→ npm run dev -- --port 3001

Issue: Blank page on browser
→ Check console for errors (F12 key)

Issue: Changes not appearing
→ Clear cache (Ctrl+Shift+Delete) and refresh

Issue: Dependencies missing
→ Run: npm install
