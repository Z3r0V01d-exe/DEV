❌ TROUBLESHOOTING:

PROBLEM: "npm run dev" shows errors
SOLUTION: 
  1. Delete node_modules folder:
     rm -r node_modules
  2. Delete package-lock.json:
     rm package-lock.json
  3. Reinstall:
     npm install
  4. Try again:
     npm run dev

────────────────────────────────────────────────────────────────────────────

PROBLEM: Vanilla version doesn't load CSS/JS with file:// protocol
SOLUTION:
  Option A: Use a local server (recommended)
    - Download Live Server extension in VS Code
    - Right-click static/index.html → "Open with Live Server"
  
  Option B: Use simple Python HTTP server
    - Open terminal in static/ folder
    - Type: python -m http.server 8000
    - Open: http://localhost:8000

────────────────────────────────────────────────────────────────────────────

PROBLEM: Port 3000 already in use
SOLUTION:
  1. Find process using port 3000:
     netstat -ano | findstr :3000
  2. Kill the process:
     taskkill /PID <PID> /F
  3. Try npm run dev again

────────────────────────────────────────────────────────────────────────────

PROBLEM: Can't find npm or Node.js
SOLUTION:
  1. Check if Node.js is installed:
     node --version
  2. If not installed, download from:
     https://nodejs.org/
  3. Install it and restart terminal
  4. Try npm install again
