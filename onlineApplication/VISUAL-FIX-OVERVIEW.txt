📁 FOLDER ORGANIZATION LOGIC:

static/                ← Vanilla JS (Standalone)
├─ index.html         ← Entry point
├─ css/
│  └─ style.css       ← All CSS in one place
└─ js/
   └─ script.js       ← All JS in one place

WHY? Separates concerns. Easy to find and modify specific file types.

public/               ← React public files
└─ index.html         ← Root element for React app

src/                  ← React source code
├─ main.jsx           ← Entry point (mounts App to #root)
├─ App.jsx            ← Main component
└─ components/        ← Form components
